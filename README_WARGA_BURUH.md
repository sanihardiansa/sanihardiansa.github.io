## 📁 File yang Dihasilkan

1. **`warga_buruh.json`** - Data lengkap dalam format JSON
2. **`warga_buruh.csv`** - Data dalam format CSV untuk analisis spreadsheet
3. **`laporan_warga_buruh.html`** - Laporan interaktif dalam format HTML
4. **`extract_buruh.py`** - Script Python untuk ekstraksi data

## 🚀 Cara Menggunakan

### Melihat Laporan:
Buka file `laporan_warga_buruh.html` di browser untuk melihat laporan interaktif.

### Analisis Data:
- Gunakan `warga_buruh.csv` untuk analisis di Excel/Google Sheets
- Gunakan `warga_buruh.json` untuk integrasi dengan sistem lain

### Menjalankan Ulang Analisis:
```bash
python3 extract_buruh.py
```
